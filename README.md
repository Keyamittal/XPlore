# XPlore 🚀

XPlore is a productivity and self-growth web application that helps users complete daily tasks, gain XP points, and stay motivated through fun daily challenges.

## Features
- Daily task management
- XP reward system
- Fun challenges after completing tasks
- Responsive UI
- Progress tracking
- Clean and interactive design

## Tech Stack
- React.js
- Tailwind CSS
- JavaScript
- Node.js

## Status
🚧 Project is currently under development.

## Future Improvements
- Authentication system
- Leaderboard
- Streak tracking
- AI-based task suggestions
